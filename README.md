## VS Code specific configurations
[Auto aprove global](vscode://settings/chat.tools.global.autoApprove) - disables the approval ask globally (DANGEROUS!)
[Auto approve terminal](vscode://settings/chat.tools.terminal.autoApprove ) - disables the approval for terminal operations
[Use Instructions files](vscode://settings/github.copilot.chat.codeGeneration.useInstructionFiles) - activates instructions, should be enabled by default
